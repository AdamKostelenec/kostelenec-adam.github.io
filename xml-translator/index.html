Shoptet XML Translator

C# · .NET · REST API · XML · 2024

Popis projektu

Shoptet XML Translator je interní nástroj pro automatizovaný překlad produktových XML feedů do cizích jazyků. Aplikace zpracovává rozsáhlé XML struktury a pomocí externího API provádí překlad jednotlivých polí (název, popis, varianty).

Cílem projektu bylo výrazně snížit časovou náročnost ruční lokalizace produktů a eliminovat chyby vznikající při manuálním překladu.

Použité technologie

  • C# / .NET – jádro aplikace
  • REST API – komunikace s překladovou službou
  • XML – parsování a transformace feedů
  • Shoptet – cílová e-commerce platforma

Funkce

  • Načtení XML feedu z URL nebo souboru
  • Detekce jazykových mutací
  • Automatický překlad vybraných elementů
  • Zachování struktury XML
  • Export hotového feedu pro import do Shoptetu

Výsledek

Nástroj je nasazen v produkčním prostředí a denně šetří hodiny ruční práce. Díky modularitě lze snadno přidat další jazyky nebo upravit pravidla překladu.
